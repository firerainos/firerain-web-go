# Install
- go get github.com/firerainos/firerain-web-go
- cd your GOPATH/github.com/firerainos/firerain-web-go
- go build main.go
- cp config.default.json config.json
- edit config.json

# Boot
`main [-p port]`

`./main` 

or 

`./main -p 80`
